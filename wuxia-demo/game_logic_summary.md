# 琅嬛福地 (秘境寻宝) 机制揭秘全览

本表单旨在帮您厘清目前已实装的所有秘境事件，包括其底层判定、出现文本及其所对应的惩罚与奖励分布，方便您进行宏观数值统筹。

## 一、 六大核心试炼机制

秘境中有 6 套底层判定机制，每次抽牌时会从每套机制对应的 3 个文案中随机抽取 1 个展示。选项不仅具有极高的挑战风险，背后还隐藏着高智力角色的隐藏彩蛋。

### 1. 残局博弈 (智力+幸运校验)
*主打脑力博弈，失败受重伤。*
* **文案变化池**：
  1. 石亭中未完的黑白残局…
  2. 石墙上一座流淌赤红真气的八卦阵图…
  3. 几卷互相牵扯的奇妙内力简卷…
* **选项判定**：
  * **【A. 安稳不惹事】** (`收敛心神`等): 必定成功。`深度 +1`，`业力 +1`。
  * **【B. 强行推演】**: 
    校验 `智慧 + 幸运 >= 玩家等级 * 1.0 + 8`。
    成功：`深度 +3`，`业力 -1`。
    失败：染上【心魔劫】被震出。
  * **【C. 天机（隐藏选项）】**: *仅当 智慧 >= 等级 * 1.5 + 8 时现形*。必定成功。`深度 +3`，`业力 +1`。

### 2. 献祭奉献 (自身血量剥夺)
*考验是否愿意舍己救人换取天道因果。*
* **文案变化池**：
  1. 腹部流血求救的雪白猿猴…
  2. 身中奇毒命不久矣的高僧…
  3. 即将尸变需要镇魂的女尸…
* **选项判定**：
  * **【A. 损耗自身气血施救】**: 强制扣除当前 HP 的 50%。若剩余血量已枯竭则直接失败染上【血枯劫】；否则`深度 +2`，`业力 +3`！
  * **【B. 强闯不留情】**: 具有 40% 随机概率踩雷失败染上【散功劫】。若拼脸成功直接强行过关 `深度 +3`，`业力 -2`。

### 3. 绝力破坏 (力量+体质联合校验)
*力拔山兮气盖世。*
* **文案变化池**：
  1. 需要掀翻的诡异金身佛像…
  2. 极重且堵死通道的断龙石…
  3. 大量阻挡前路的铁甲地衣蟹…
* **选项判定**：
  * **【A. 忍辱退避】** (`钻侧缝/爬墙`等): 必定成功。`深度 +2`，`业力 +1`。
  * **【B. 硬生生轰碎】**:
    校验 `力量 + 体质 >= 玩家等级 * 1.5 + 10`。
    成功：霸道过关！`深度 +3`，`业力 -2`。
    失败：被暗劲反伤，染上【血枯劫】大口呕血踢出秘境。

### 4. 遗迹承恩 (高体质灵力感应)
*考察有无仙缘接受前辈造化。*
* **文案变化池**：
  1. 剑冢枯骨留下的绝笔字迹…
  2. 寒泉底部的森寒仙女舞剑图…
  3. 蒲团旁未熄灭且异香扑鼻的古铜香炉…
* **选项判定**：
  * **【A. 恭敬祭奠】**:
    校验 `体质 >= 玩家等级 * 1.0 + 8`。
    满足条件：强悍体魄引动真气感悟！`深度 +3`，`业力 +3`。
    未满条件：单纯拜别。`深度 +2`，`业力 +1`。
  * **【B. 蛮横抢夺造化】**:
    冒犯前人，有 40% 随机概率触发禁制染上【心魔劫】。若脸白幸存，强力冲关 `深度 +4`，`业力 -3`。

### 5. 幻境审视 (心境与武学信念校验)
*直击内心最深处的欲望。*
* **文案变化池**：
  1. 引诱你交出自身皮囊的冰魔之镜…
  2. 桃花林中唾手可得的神兵与财宝…
  3. 被过往仇杀厉鬼包围的修罗血海…
* **选项判定**：
  * **【A. 坚守本心出尘】**:
    校验 `智慧 + 幸运/2 >= 玩家等级 * 1.2 + 8`。
    成功：道心通明，`深度 +3`，`业力 +1`。
    失败：定力不足崩溃出局。染上【心魔劫】。
  * **【B. 以暴制暴斩落虚妄】**:
    校验 `力量 >= 玩家等级 * 1.5 + 15`。
    成功：以杀证道，`深度 +3`，`业力 -2`。
    失败：无力撼动，精神震荡直接无惩罚出局(不染Debuff，但探险停止)。

### 6. 轻巧之极 (极致敏捷要求)
*天下武功唯快不破。*
* **文案变化池**：
  1. 满是连弩夺命陷阱的甬道…
  2. 狂风大作极为湿滑的深渊断桥…
  3. 即将淹没头顶的流沙密室千钧一发…
* **选项判定**：
  * **【A. 狼狈逃脱】**: 虽然慢但必定安稳。`深度 +1`，`业力 +0`。
  * **【B. 秀操作身法过关】**:
    严格校验 `敏捷 >= 玩家等级 * 2.0 + 8`。
    成功：如鬼魅般轻松过关，`深度 +3`，`业力 +0`。
    失败：身手跟不上思维，重伤坠崖，染上【血枯劫】出局。

## 二、 门槛奖励表 (深度与业力机制)

如果在探险中点击了**【见好就收，遁出秘境】**或正常耗尽事件卡组完成探险，系统会根据积累的**最终深度 (Depth)** 和 **因果业力 (Karma)** 来发放奖励。

### 奖励层级：
| 最终深度达标 | 解锁奖池稀有度 | 对应奖励分布描述 |
|:---:|:---:|:---|
| **0 ~ 4层** | 无 | *“浅尝辄止，一无所获”* - 无法带走任何宝具，安全白跑。 |
| **5 ~ 9层** | `普通`，`稀有` | 基础保底，能抽中诸如【木质佛珠】(普通)、【白玉短笛】(稀有)一类。 |
| **10 ~ 14层** | `稀有`，`史诗` | 豪华奖池阶段，必得紫装及以上。**此阶段享受极端的业力过滤法则（见下方详情）**。 |
| **15层极境** | `史诗`，`传说`，`神话` | 只有依靠高属性连连过关甚至天机卡才能到达此境！绝世神器【屠龙刀】、【绝世好剑】仅在此处出炉！ |

### 特殊因果过滤 (Karma 干扰)
在 **10 ~ 14层** 阶段，玩家结出的善恶之树会直接主导装备倾向：
* **善果极大 (业力 >= 5)**：奖池进一步收窄为**防守生存型**。只会掉落 `['回春', '荆棘(软猬甲)', '倚天(吸血)', '轻巧']` 这四类特效神兵！
* **恶果极大 (业力 <= -5)**：奖池进一步收窄为**暴虐嗜血型**。只会掉落 `['破伤', '点穴', '剧毒', '打狗(击晕)', '金蛇(连击)']` 这五类极具攻击性神兵！

> [!NOTE]
> 鼓励玩家明确自己的构建套路：玩叠甲流就必须尽量以慈悲心（高 Karma）过关止步 10+ 层；玩毒刺流就得一路强行破拆损人利己（低 Karma）到达 10+ 层！这种挂钩真正把叙事选择和游戏性黏合在了一起。
