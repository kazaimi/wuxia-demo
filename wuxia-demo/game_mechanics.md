# 武侠 Demo - 战斗机制与隐藏数值白皮书

本文档提权了当前游戏所有底层战斗逻辑、武学判定概率及宝物隐藏属性，以供开发参考或高玩进阶使用。

## 一、 基础战斗公式
1. **出手顺序（速度判定）**
   - 机制：每回合基于双方敏捷值摇骰子。敏捷越高，先手抢攻概率越高。
   - 概率公式：`攻击方敏捷 / (攻击方敏捷 + 防守方敏捷 + 1)`

2. **面板基础攻防**
   - 基础攻击力：`力量 × 2 + 等级 × 5`
   - 基础防御力：`体质 × 2 + 等级 × 2`

3. **伤害结算**
   - 武学实际威力：`技能基础威力 × (1 + 等级 × 0.05)`
   - 最终受到伤害：`Max(1, 基础攻击力 + 武学实际威力 - 最终防御力 × 1)`
   - *（注：防守方防御力在计算伤害时只会生效 50%，除非拥有“防御增加”的 Buff，此时防御力抗伤效果翻倍，即抵消更多伤害）*

4. **闪避判定 (轻功)**
   - 基础无 Buff 闪避率：`防守方敏捷 × 0.5%`
   - 极限闪避率：当身上拥有【闪避增益】Buff 时，闪避几率强制锁定为 **45%**。

## 二、 武学隐藏特效与施法概率

绝大多数进攻武学的判定只是简单的伤害结算，但部分特定内功、绝学及轻功带有隐藏附加特效：

- **凌波微步 / 梯云纵**：释放后获得 3 回合【大幅闪避增益】（该状态下闪避率固定为 45%）。
- **神行百变**：释放后获得 **99 回合**【极限闪避增益】（基本等同于整场战斗闪避维持在 45%）。
- **九阳神功**：释放后获得 3 回合【真气护体】（防御力翻倍）。
- **易筋经**：释放后获得 3 回合【真气护体】（防御力翻倍），且**瞬间清零身上所有的剧毒层数**！
- **圣心诀**：释放后获得【涅槃重生】Buff（持续1局，当血量归零时以 50% 最大生命值原地复活一次）。
- **吸星大法**：吸血机制，将该次技能造成伤害的 **80%** 转化为自身气血恢复。
- **千蛛万毒手**：命中后对敌方附加永久【剧毒】（999回合标识）。剧毒每回合固定扣除敌方 **最大生命值的 3%**。
- **狮吼功**：技能触发后有 **60% 概率**判定敌方【晕眩】（1回合无法物理行动）。
- **葵花点穴手**：技能触发后有 **80% 概率**判定敌方【封穴】（2回合无法使用任何主动武学，由于经脉闭塞只能使用基础拳脚）。

*(注：控制类状态无法叠加，每次重复触发只刷新持续时间。)*

## 四、 奇遇系统掉率隐藏机制
1. **宝物掉落率**：
   - 玩家在【江湖奇遇】连战模式中，必须连续击败 3 名真实玩家镜像或 NPC 才可进入战利品结算环节。
   - 掉落判定的来源池为：所击败的 3 个敌人的身上实际佩戴宝具。
   - **隐藏掉率公式**：基础掉率为 5%。玩家每多 1 点【幸运】属性，掉率增加 1%。
     即 `5% + (幸运值 × 1%)`。
   - 掉率存在硬性上限，单次判定最高封顶阀值为 **25%**。
2. **经验收益补正**：
   - 战胜高榜单排名或高等级的对手时，结算经验将获得巨额的乘区放大收益。
