# 1.4版本「软猬甲」更新评估与宝具设计

## 1. 需求评估报告
阅读了您的 1.4 版本更新文档，整体设计非常精彩，大大加深了游戏的策略维度和重玩价值：

1. **界面与洗点优化**：增加版本号标识，洗点支持手动输入，这是极大的用户体验（UX）提升。
2. **功法槽位系统**：将武学细分为`内功`、`外功`、`身法`、`绝学`四个独立槽位。这也意味着我们需要对现有的 `SKILLS_DB` 进行分类扩展，确保玩家有足够的不同类型武术可以选择。
3. **宝具与特效系统**：这是本版本的核心重头戏。引入宝具（附加属性+战中被动特效）会让战斗过程不再是单纯的数值碰撞。中毒、击晕、吸血、反伤、封印等状态机（Buff/Debuff系统）需要集成到现有的 Socket 战斗结算循环中。
4. **NPC更新规划**：按排名给五十名之前的NPC分配宝具，并为所有的NPC配置专门的功法槽。高排名的大魔王将会有极高的挑战难度。
5. **江湖奇遇模块（小队连战）**：这是一个非常有意思的 Rogue-like 机制！玩家血量翻倍但不恢复，连续挑战三人，胜利有概率掉落神兵利器（概率由幸运值决定），每天限制5次，为《江湖风云榜》增加了极强的PVE可玩性核心循环。
